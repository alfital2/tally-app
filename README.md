# Tally — landing page

The marketing site + release host for **Tally**, a tiny macOS menu-bar app that
shows your Claude.ai session and weekly usage limits in real time, as
Apple-Watch-style rings.

- **Site:** https://alfital2.github.io/tally-app/
- **Download:** [latest release](https://github.com/alfital2/tally-app/releases/latest) (signed + notarized `.dmg`)

The app source lives in a separate repository. Tally is an independent project
and is not affiliated with or endorsed by Anthropic.
